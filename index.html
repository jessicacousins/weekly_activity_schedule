Weekly Schedule Tracker

WARNING: Demo Only

This live front-end demo showcases a MERN-based scheduling platform built for coordination, billing, attendance, and performance insights.

Note: Data is stored only in your browser. This demo is for testing and planning — not for sensitive information.

Quick Start Guide

  • ★ Create employees using the “New Employee” button.
  • ★ Click an employee to edit their schedule or attendance.
  • ★ Click a client in the schedule to enter/view daily notes.
  • ★ Manager Dashboard: View top clients, attendance heatmaps, and export reports.
  • ★ PDF Export: Download schedules, notes, and dashboards with one click.

This tool is ideal for planning and internal coordination — no cloud, no accounts, no tracking.
