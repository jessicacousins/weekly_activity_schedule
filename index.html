Weekly Schedule Tracker - Demo

This is a live front-end prototype of a fullstack MERN-based scheduling platform—designed for coordination, billing, attendance tracking, and staff performance insights.

Important: All data is stored in your browser only (no backend, no cloud). This demo is not intended for storing sensitive information.

It’s an early look at a scalable staff management system tailored for care teams, coordinators, and workplace dashboards.

How to Use This Tool

This website allows you to create and manage weekly schedules and attendance records for your employees. You can:

  • ★ Create and manage employees using the "Create New Employee" button.
  • ★ Click on an employee’s name to enter their weekly schedule or attendance data.
  • ★ Click on a client’s name in the schedule view to enter daily comments for that client. You can review past notes in a calendar-style format and download them as a printable PDF—either for a full history or a selected date range.
  • ★ View a detailed overview in the Manager Dashboard, including client counts, attendance heatmaps, and top client stats.
  • ★ Use the “Download Dashboard PDF” button on the dashboard to export all visual reports, weekly summaries, and attendance data into a printable PDF.

Note: All data is stored locally in your browser. This tool is designed for lightweight planning and should not be used to store sensitive personal information.
